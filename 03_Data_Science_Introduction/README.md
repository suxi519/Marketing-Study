# 데이터 과학이란?  
* 데이터 마이닝과 유사하게 정형, 비정형 형태를 포함한 다양한 데이터로부터 지식과 인사이트를 추출하는데 과학점 방법론, 프로세스, 알고리즘, 시스템을 동원하는 융합분야.
