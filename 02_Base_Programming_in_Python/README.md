### 3. 자료형(data type)  

#### 3-1. 숫자(number)  
* 정수(integer) 2  
* 소수(floating point) 2.0    

#### 3-2. 문자열(string)  
* 키보드로 쓸 수 있는 문자들을 표현하는 자료형  
* "2" or '2'  

형 변환(Type Conversion)   
값을 한 자료형에서 다른 자료형으로 바꾸는 것       

정수 7 -> 소수 7.0    
문자역 "7" -> 정수 7    
````python
print(int(3.8))  
````

### 4.추상화(abstraction)  
#### 4-1. 변수(variable)  
* oreo = 990  

#### 4-2. 함수(function)
```python
def hello():
    print("hello")
    print("welcome to korea")

hello()
```

#### 4-3. 파라미터(parameter)  
```python
def hello(name):
    print("hello")
    print(name)
    print(welcome to korea)

hello("sohee")
```

#### 4-4. 여러 개의 파라미터  
```python
def print_sum(a, b, c):
    print(a + b + c)

print_sum(7, 3, 1)

```
